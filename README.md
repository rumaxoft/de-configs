## git

- .themes.gitconfig

##### path:

##### install delta ([repo](https://github.com/dandavison/delta))
 - Fedora
```sh
dnf install git-delta
```
- Debian 
```sh
dpkg -i file.deb
```
.deb files are on the [releases page](https://github.com/dandavison/delta/releases).
